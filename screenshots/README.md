# Screenshots Directory

Add screenshots of the Jambo Bank application here.

## Files to add:
- `jambo_bank_pro_home.png` - Welcome screen of the professional version
- `jambo_bank_pro_dashboard.png` - Dashboard after login
- `jambo_bank_pro_deposit.png` - Deposit transaction screen

## How to add:
1. Run the application: `python jambo_bank_pro.py`
2. Take screenshots of key screens using:
   - Windows: Press `Win + Shift + S`
   - Save as PNG files in this directory
3. Commit and push to GitHub

Example:
```bash
git add screenshots/
git commit -m "Add application screenshots"
git push
```
